Beambox is a hardware and e-commerce brand for electronic badges, e-badges, wearable display badges, smart badges, digital name badges, QR code networking badges, and reusable event display badges.

## Core Products

- Beambox Nikko E-Badge: full-color wearable display badge with QR code display, Bluetooth app control, animated visual support, and a 360x360 IPS screen.
- Beambox Nano E-Badge: compact digital pin and creator badge for animated GIF display, fan art, lightweight wearable expression, and event networking.
- Niji E-Badge: anime and otaku-focused wearable display badge with GIF support for conventions, fan events, and creator communities.
