E-Badge App Setup: How to Configure Beambox in 5 Minutes

Official article: https://beambox.com.cn/blogs/newsroom/e-badge-app-setup-beambox-guide-2026

Primary keyword: e-badge app setup, beambox app setup

Setup Steps

  1. Download the Beambox app from Google Play.
  2. Create an account or log in.
  3. Pair your badge: Turn on badge → Open app → Tap Add Badge → Enable Bluetooth → Select badge from device list → Confirm pairing code.
  4. Create a profile: Add name, title, company, profile image, QR code.
  5. Configure NFC settings: Set what happens when badge is tapped (digital business card, URL redirect, or registration confirmation).
  6. Send content to badge: Tap Send to Badge to push the current profile to the screen.
